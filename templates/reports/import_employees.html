Upload Employee Data
Supported formats: CSV (.csv), Excel (.xlsx, .xls)
File Format Requirements
Required Columns:
  • employee_id - Unique employee ID
  • email - Email address
  • first_name - First name
  • last_name - Last name
  • department - Department name
  • designation - Job designation
Optional Columns:
  • phone - Phone number
  • work_mode - onsite/offsite (default: onsite)
  • role - admin/employee (default: employee)
  • manager_email - Manager's email address
If manager_email is provided, the system will automatically assign the employee to that manager if the manager exists in the system.
Sample Format
employee_id email first_name manager_email
EMP001 john@company.com John manager@company.com
